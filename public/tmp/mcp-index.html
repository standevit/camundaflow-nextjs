Model Context Protocol (MCP)

Ein aufkommender offener Standard, der es KI-Agenten ermöglicht, externe Tools und andere Agenten sicher zu entdecken, zu verstehen und zu nutzen — das „USB-C der Agenten-Interoperabilität", das im Multi-Agenten-Ökosystem schnell an Bedeutung gewinnt.

Was ermöglicht MCP konkret?

  • Entdeckt verfügbare Tools automatisch (keine hart-codierten Integrationen)
  • Versteht, was jedes Tool tut, seine Parameter und erforderliche Berechtigungen
  • Ruft jedes MCP-kompatible Tool auf (Websuche, Dateisystem, GitHub, Datenbanken, Rechner, Wikis, APIs…)
  • Funktioniert framework-übergreifend — LangChain, CrewAI, Autogen, LlamaIndex, Semantic Kernel und viele andere unterstützen oder planen MCP bereits
  • Unterstützt Human-in-the-Loop — sensible Aktionen können vor der Ausführung eine explizite Benutzerfreigabe erfordern
  • Ermöglicht Multi-Agenten-Zusammenarbeit via A2A (Agent-to-Agent) über dasselbe Protokoll

Typische Tools, die heute via MCP verfügbar sind (2025/2026)

  • Fetch URL / Web Browser Tools
  • Dateisystem-Lese-/Schreiboperationen
  • GitHub (Issues, PRs, Repo-Zugriff)
  • DeepWiki / interne Wissensdatenbank-Suche
  • Zeit- und Datumskonvertierungen, Mathematik-Rechner
  • Beliebige REST/GraphQL-Services als MCP-Tool verpackt
  • Datenbank-Query-Tools (SQL, Vektor-DBs)

Warum Camunda 8 + MCP die perfekte Enterprise-Kombination ist

  • 100% Transparenz & Nachvollziehbarkeit — jeder MCP-Tool-Aufruf ist ein regulärer BPMN-Service-Task
  • Eingebaute Leitplanken — DMN-Tabellen entscheiden, welche Tool-Aufrufe eine Benutzerbestätigung benötigen
  • Keine Black-Box-Agenten — vollständige Nachvollziehbarkeit, was der Agent gesehen, entschieden und ausgeführt hat
  • Human-in-the-Loop jederzeit möglich — Eskalation zur Tasklist mit einem Klick
  • Agenten mischen & kombinieren — Camunda-Agenten neben OpenAI, Anthropic, Mistral oder jedem anderen MCP-kompatiblen Agenten verwenden
  • Skaliert auf Millionen von Tool-Aufrufen pro Tag auf der Zeebe Cloud-Native-Engine

Fazit: MCP ist nicht nur ein weiteres Integrationsformat — es ist der universelle Stecker, der es Unternehmen endlich ermöglicht, KI-Agenten sicher und kontrolliert echte Power zu geben, indem sie mit Hunderten von Tools verbunden werden, während alles transparent, nachvollziehbar und unter voller Prozess-Orchestrierungs-Kontrolle mit Camunda bleibt.
